# Autho.R - Pricing Tiers

## Overview

Autho.R offers five pricing tiers designed to scale with your automation needs. All plans include access to our web-based UI and core automation capabilities.

## Pricing Summary

| Tier | Monthly | Annual | Tasks/Month | Modules | Support |
|------|---------|--------|-------------|---------|---------|
| **Free** | $0 | $0 | 100 | 4 | Community |
| **Starter** | $29 | $290 (save 17%) | 1,000 | 7 | Email |
| **Professional** | $79 | $790 (save 17%) | 10,000 | 10 | Priority |
| **Business** | $199 | $1,990 (save 17%) | 50,000 | 10+ | Dedicated |
| **Enterprise** | $499+ | $4,990+ | Unlimited | All | Dedicated + SLA |

## Tier Details

### Free - $0/month
*Get started with basic automation*

**Limits:**
- 100 tasks per month
- 5 workflows per month
- 50 API calls per day
- 10 MB max file size
- 7-day data retention

**Modules Included:**
- Spreadsheet (CSV/Excel)
- Files (read/write/organize)
- PDF (extract, merge, create)
- Documents (Word, Markdown)

**Features:**
- Community support via GitHub
- Web UI access

## Marketplace Availability

### Azure Marketplace
- SKU: `author-free`, `author-starter`, `author-professional`, `author-business`, `author-enterprise`
- Deployment: ARM Template
- Billing: Monthly or Annual via Azure

### IBM Cloud Marketplace
- Catalog ID: `author-framework`
- Deployment: Cloud Foundry
- Billing: Monthly via IBM Cloud
